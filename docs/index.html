A wise codex of what your computer just heard

Hibiki Codex captures your system audio, transcribes it locally with whisper.cpp in Japanese or English, and lets you ask Claude or Codex about what you've been hearing — all in one Windows desktop app.

View on GitHub → Get started

Three steps, one window.

The whole loop runs locally on your machine — only your prompt and the surrounding transcript are sent to Claude or Codex when you ask.

一

Capture

WASAPI loopback grabs whatever your default playback device is playing — meetings, lectures, anime, anything.

二

Transcribe

whisper.cpp runs locally with a rolling buffer, streaming chat-style transcript lines in Japanese or English.

三

Ask

Send a prompt to Claude, Codex, or both side-by-side. The recent transcript is attached as context automatically.

Built on tools you trust.

A small, sharp stack — React on the surface, Electron and Rust underneath, whisper.cpp doing the heavy lifting offline.

Get started.

Hibiki Codex runs on Windows 11 with PowerShell 7+. Toolchain versions are pinned in mise.toml, so setup is three commands. 

# 1. Install mise (one-time)
winget install jdx.mise

# 2. Install Bun + Rust at the pinned versions
mise install
mise trust

# 3. Install JS deps and build the Rust native module
bun install
bun install --cwd native
bun run build:rust

# Then point Settings at whisper-cli.exe + a ggml-*.bin model and press Start.
bun run dev

Prefer a one-shot install? Paste the Setup with AI prompt from the README into Claude Code or Codex — it installs every prerequisite, downloads whisper.cpp + a model, and writes the settings file for you.
